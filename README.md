# Portfolio 2026

Este repositório contém o código-fonte do meu portfólio pessoal para 2026.

## Estrutura do Projeto

- `index.html`: Página principal do portfólio.
- `style.css`: Estilos utilizados no site.
- `images/`: Imagens utilizadas no portfólio.
- `stitch-layout.png`: Planejamento inicial do layout do portfólio.

## Como visualizar

Você pode visualizar o portfólio diretamente pelo GitHub Pages:

- <https://danielarjordao.github.io/portfolio-2026/>

Ou, se preferir, clone o repositório e abra o arquivo `index.html` em seu navegador.
